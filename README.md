1. Uso de Cookies

Las cookies son pequeños datos que un sitio web almacena en el navegador del usuario. Su función es recordar información importante, como el nombre de usuario, preferencias o configuraciones, para mejorar la experiencia de navegación.

2. Creación de Sesiones

La creación de sesiones permite que un usuario permanezca autenticado mientras utiliza una aplicación web. De esta forma, puede navegar entre diferentes páginas sin tener que iniciar sesión cada vez.

3. Eliminación de Sesiones

La eliminación de sesiones consiste en cerrar la sesión del usuario y borrar los datos temporales asociados. Esto ayuda a proteger la privacidad y evita accesos no autorizados, especialmente en equipos compartidos.
